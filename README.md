## Overview

A corporate website for ESR (Engineering & Sustainable Research) or a similar organization. It includes pages for services, portfolio, testimonials, blog and contact information.

## Features

| ✔️ | Feature |
|---|---|
| ✅ | Home page introducing the organization |
| ✅ | Services page listing offerings |
| ✅ | Portfolio gallery showcasing projects |
| ✅ | Contact page with form and contact details |


## Tech Stack

- 🛠️ HTML
- 🛠️ CSS
- 🛠️ JavaScript
- 🛠️ SCSS

## Installation

Follow these steps to get a local copy of the project up and running:

1. Clone the repo: `git clone https://github.com/Nischhalsubba/ESR-Website.git`.
2. Open the site locally using a static server or just open the HTML files.

## Usage

Customize the content to reflect your company and deploy via GitHub Pages or any hosting provider.
